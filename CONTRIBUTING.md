# Cómo contribuir

1. ¡Hazle un fork!
2. Crea una rama para tu desarrollo: `git checkout -b mi-nuevo-desarrollo`
3. Confirma tus cambios: `git commit -m 'Add some feature'`
4. Envíalos a la rama: `git push origin mi-nuevo-desarrollo`
5. Envía un pull request

**Después de que tu pull request sea fusionado**

Después de que tu pull request sea aceptado y fusionado, puedes borrar con seguridad tu rama.
